# :dolls: Pop in a box shop :dolls:

simple e-commerce shop with JavaScript and JQuery:

## Technologies used

- Javascript
- JQuery
- Jasmine
- HTML & CSS

## How to run it

```
git clone git@github.com:GBouffard/pop-in-a-box-shop.git
cd pop-in-a-box-shop
npm install
```
